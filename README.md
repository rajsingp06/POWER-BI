HR Analytics Dashboard using Power BI

This project is developed as part of the Microsoft LMS Power BI Internship Program. The goal of this project is to analyze employee data and create an interactive HR Analytics Dashboard that helps organizations understand workforce trends and make data-driven decisions.

The dashboard visualizes important HR metrics such as employee distribution, attrition rate, department analysis, salary insights, and job satisfaction using powerful data visualization techniques in Power BI.

By using interactive charts, KPI cards, and filters, the dashboard allows HR managers to easily explore employee data and identify patterns related to employee retention and workforce management.

Key Features

Employee distribution analysis

Attrition count and attrition rate visualization

Department-wise employee insights

Salary and job satisfaction analysis

Interactive dashboard with filters and visual charts

Tools & Technologies

Microsoft Power BI

Microsoft Excel Dataset

DAX (Data Analysis Expressions)

Data Visualization Techniques

Project Outcome

The HR Analytics Dashboard provides meaningful insights into employee data and helps HR teams improve decision-making through visual analytics.
